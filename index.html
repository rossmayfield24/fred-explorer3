One-time setup needed

Enter your FRED API key and Cloudflare Worker URL in Steps 1 and 2 in the sidebar to get started.

New user? Set up your proxy in 5 minutes:

1. Go to workers.cloudflare.com → free account → Create Worker → Start with Hello World
2. Click Deploy, then Edit code
3. Select all code (Ctrl+A), paste the code below, click Save and deploy
4. Copy your Worker URL and paste it into Step 2 in the sidebar

Worker code: 
export default {
  async fetch(request) {
    if (request.method === 'OPTIONS') {
      return new Response(null, { headers: {
        'Access-Control-Allow-Origin': '*',
        'Access-Control-Allow-Methods': 'GET, OPTIONS',
        'Access-Control-Allow-Headers': '*'
      }});
    }
    const url = new URL(request.url);
    const path = url.pathname.replace(/^\/fred\//, '');
    const params = url.searchParams.toString();
    const fredUrl = `https://api.stlouisfed.org/fred/${path}${params ? '?' + params : ''}`;
    const resp = await fetch(fredUrl, { headers: { 'User-Agent': 'FRED-Explorer/1.0' } });
    const body = await resp.text();
    return new Response(body, { status: resp.status, headers: {
      'Content-Type': resp.headers.get('Content-Type') || 'application/json',
      'Access-Control-Allow-Origin': '*',
      'Cache-Control': 'public, max-age=300'
    }});
  }
};
