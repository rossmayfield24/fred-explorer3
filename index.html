One-time setup needed

Because the FRED API blocks direct browser requests, this tool uses a free personal proxy you control. It takes about 5 minutes to set up once.

Setup steps:

1. Go to workers.cloudflare.com and sign up for a free account (no credit card needed)
2. Click "Create application" → "Create Worker"
3. Delete all the default code in the editor, paste in the Worker code below, then click "Deploy"
4. Copy the Worker URL shown (looks like fred-proxy.yourname.workers.dev)
5. Paste it into Step 2 in the sidebar, click Save

Worker code to paste: 
export default {
  async fetch(request) {
    if (request.method === 'OPTIONS') {
      return new Response(null, { headers: {
        'Access-Control-Allow-Origin': '*',
        'Access-Control-Allow-Methods': 'GET, OPTIONS',
        'Access-Control-Allow-Headers': '*'
      }});
    }
    const url = new URL(request.url);
    const path = url.pathname.replace(/^\/fred\//, '');
    const params = url.searchParams.toString();
    const fredUrl = `https://api.stlouisfed.org/fred/${path}${params ? '?' + params : ''}`;
    const resp = await fetch(fredUrl, { headers: { 'User-Agent': 'FRED-Explorer/1.0' } });
    const body = await resp.text();
    return new Response(body, { status: resp.status, headers: {
      'Content-Type': resp.headers.get('Content-Type') || 'application/json',
      'Access-Control-Allow-Origin': '*',
      'Cache-Control': 'public, max-age=300'
    }});
  }
};
